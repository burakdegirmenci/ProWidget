Son olarak PrimeWidgetsX için dokümantasyon ve örnek kullanım senaryosu yazalım.

İstediğim:

1. /docs/api-reference.md
   - PUBLIC endpoint’ler:
     - GET /api/:slug/config
     - GET /api/:slug/theme
     - GET /api/:slug/data
   - ADMIN endpoint’ler:
     - Login
     - Customers CRUD
     - Widgets CRUD
     - Theme CRUD
   - Her endpoint için:
     - URL
     - Method
     - Body örneği
     - Response örneği

2. /docs/developer-guide.md
   - Projeyi local’de ayağa kaldırma:
     - Gerekli araçlar
     - docker-compose ile adımlar
   - XML feed entegrasyonu nasıl yapılır
   - Yeni widget ekleme rehberi (adım adım)

3. README.md
   - Proje özeti:
     - “Tek satır script ile çalışan, XML/JSON ile beslenen, e-ticaret widget SaaS platformu”
   - Özellikler listesi
   - Tech stack
   - Quick start

4. Örnek müşteri: Elle Shoes
   - /docs/examples/elleshoes-usage.md
     - Elle Shoes için XML örneği
     - Admin panelde nasıl tanımlanır
     - Sitede nasıl kullanır:
       - <script defer src="https://cdn.primewidgetsx.com/elleshoes/init.js"></script>
       - <div data-pwx-widget="carousel" data-category="bot" data-limit="8"></div>
       - <div data-pwx-widget="banner" data-campaign="winter20"></div>
